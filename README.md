# appropriate/tcpflow
Alpine-based image with just tcpflow

## Usage

```console
$ docker run --rm --net=host appropriate/tcpflow
```

## Tags

* `appropriate/tcpflow:latest`: based on `alpine:latest`
* `appropriate/tcpflow:edge`: based on `alpine:edge`
